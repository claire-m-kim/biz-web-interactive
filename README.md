# 🌊 Bluewave Systems - AI Assisted Web Project

Bluewave Systems는 최신 AI 도구를 디자인·개발 워크플로우에 통합하여  
**기획, 시각 설계, 구현까지의 전 과정을 실험적으로 검증한 웹 포트폴리오 프로젝트**입니다.

단순 자동 생성이 아닌,  
AI의 결과물을 **분석·선별·수정하는 디자이너/개발자의 판단 과정**에 중점을 두었습니다.

---

## 🚀 AI-Powered Workflow (Core Competencies)

본 프로젝트는 기획 단계부터 개발 및 배포까지 AI를 적극 활용하되,  
각 단계에서 사람의 판단과 수정을 통해 완성도를 높였습니다.

### 1. Visual Direction (Gemini AI)
- 프로젝트 컨셉과 브랜드 톤에 맞는 비주얼 에셋을 생성하기 위해 프롬프트를 설계했습니다.
- 생성된 결과물 중 브랜드 아이덴티티에 부합하는 이미지를 선별하고,  
  실제 웹 적용을 고려해 사용 여부를 판단했습니다.

### 2. Web Design & Architecture (Figma AI)
- Figma AI를 활용해 기본 레이아웃과 구조를 설계했습니다.
- AI가 제안한 UI를 그대로 사용하지 않고,  
  UX 흐름과 시각적 밀도를 기준으로 직접 수정·보완했습니다.

### 3. Development (Figma-to-Code)
- AI 기반 Figma-to-Code 방식을 활용해 초기 개발 속도를 향상시켰습니다.
- 추출된 코드의 구조를 분석하고,  
  React + Vite 환경에 맞게 리팩토링 및 디버깅을 진행했습니다.

---

## 💡 Insight
이 프로젝트를 통해 AI를 **결과 생성 도구가 아닌 생산성 증폭 도구**로 활용하는 경험을 했습니다.  
AI가 만든 초안을 기반으로 구조를 이해하고 개선함으로써,  
디자인과 개발 전반을 아우르는 문제 해결 역량을 강화했습니다.

---

## 🛠 Tech Stack
- **AI Tools:** Gemini (Image Generation), Figma AI (Design & Code)
- **Frontend:** React, Vite
- **Deployment:** GitHub Pages

# JP

# 🌊 Bluewave Systems - AI Assisted Web Project

Bluewave Systemsは、最新のAIツールをデザイン・開発のワークフローに統合し、  
**企画・ビジュアル設計・実装までの一連の工程を効率化・検証したWebポートフォリオプロジェクト**です。

単なる自動生成に留まらず、AIが出力した結果を  
**分析・選定・最適化する「デザイナーとしてのディレクション能力」**に重点を置いています。

---

## 🚀 AI-Powered Workflow (Core Competencies)

本プロジェクトでは、企画段階からデプロイまでAIを積極的に活用しつつ、  
各フェーズにおいて人の判断による調整を加えることで、**実務レベルの完成度**を追求しました。

### 1. Visual Direction (Gemini AI)
- ブランドコンセプトに合致したビジュアルアセットを生成するため、詳細なプロンプト設計を行いました。
- 生成された候補の中からブランドアイデンティティに最適なものを厳選し、  
  Webデザインとしての整合性を考慮して採用を判断しました。

### 2. Web Design & Architecture (Figma AI)
- Figma AIを活用し、情報構造の骨子とレイアウトのプロトタイプを迅速に構築しました。
- AIの提案をベースに、UX（ユーザーエクスペリエンス）の導線や  
  **視覚的な優先順位**を考慮し、手動でデザインのブラッシュアップを行いました。

### 3. Development (Figma-to-Code)
- AIベースのFigma-to-Code手法を採用し、コーディングの初期工数を大幅に削減しました。
- 出力されたコードの構造を解析し、  
  React + Vite環境において保守性の高いコードへとリファクタリングおよびデバッグを実施しました。

---

## 💡 Insight
本プロジェクトを通じて、AIを単なる「生成ツール」ではなく、  
**「人間の創造性を最大化させる生産性向上ツール」**として活用する経験を得ました。

AIによる初案を批判的に分析し、改善を繰り返すプロセスの中で、  
デザインとフロントエンドの両面における問題解決能力を養うことができました。

---
